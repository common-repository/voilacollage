Place interactive collages in your posts

== Description ==

[Voilà!Collage](https://voilacollage.com) is a free interactive collage maker.
This plugin allows you to easily embed collages in your posts and gives your a little more than a regular embed:
it keeps proper aspect ratio on every screen and allows users to open a full-page slideshow.

For more information go to [voilacollage.com](https://voilacollage.com) or
drop us a line at <bureau@voilacollage.com>.

Learn more about our [Terms of Service](https://voilacollage.com/terms-of-service)
and [Privacy Policy](https://voilacollage.com/privacy-policy).

== Installation ==
1. Upload the plugin files to the /wp-content/plugins/plugin-name directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the ‘Plugins’ screen in WordPress.
3. Add a Voilà!Collage block in your post content.
4. Paste your collage url in the input field.

Enjoy!

For more information go to [voilacollage.com](https://voilacollage.com) or drop us a line at <bureau@voilacollage.com>.
